# Mensageiro

Sistema interno de comunicação por e-mail com templates.

## Funcionalidades

- Cadastro e login de usuários
- Criação de templates de e-mail
- Envio de mensagens usando templates
- Caixa de entrada e histórico
- API protegida com JWT

## Tecnologias

- Python
- Flask
- SQLite
- Docker

## Executar localmente

```bash
python app.py
